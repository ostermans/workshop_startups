В проекте разработана модель машинного обучения с использованием пайплайна для предсказания статуса деятельности стартапа. 

Проведена обработка данных, разведочный анализ, обнаружены признаки влияющие на успешность стартапа.

Корреляционным анализом исключили переобучение.
Лучшей метрикой на тренировочном датасете 0.95. На тестовом датасете добились максимальной оценки 0.67

Анализ признаков показал наиболее важными признаками признаки категорий, стран и объема финансирования.
